# Jinx OS
Jinx is a basic operating system built from scratch.

## Install
You can run Jinx OS in Docker.

### Windows
```batch
git pull https://github.com/matthewflegg/open-jinx.git
cd open-jinx
.\buildenv\run.bat
```

### Linux
```bash
git pull https://github.com/matthewflegg/open-jinx.git
cd open-jinx
chmod +x ./buildenv/run.sh
sudo ./run.sh
```
